Requisitos Minimos BSCM Link

Banco de Dados

Fazer o OWNER para BSCM com permissões e sinônimos nas tabelas. Ter na máquina o TNSNAMES configurado juntamente com o PLSQL instalado.

Programas Necessários

  1. .NET Core 5.4
  2. .Net Framework 4.8
  3. .Net 5.0.4
  4. Asp Net 5.0.6
  5. ASP NET Core Runtime
  6. ASP.NET Core Hosting

Requisitos da Maquina

  1. Processador: Quad-Core de 2.4 ou superior
  2. Memória Mínima: 8GB
  3. Disco Rígido (HD): 10GB Disponível
  4. Sistema Operacional Mínimo: Windows 7 ou Superior
  5. Internet mínima: 2 Mbps
  6. Portas liberadas: 10500 até 10550
  7. Já deixar configurado: IIS Microsoft Internet Information Services versão 7 ou superior (habilitar ASP.NET, ASP.NET Core)
